# Byte Bit Tool (BBT)

Convert numbers between Decimal, Hexadecimal and Binary with support for arithmetic and bitwise operations directly in VS Code!

## Features

- 🔢 **Number Conversion** - Convert between DEC, HEX, and BIN instantly
- ➕ **Arithmetic Operations** - Support for +, -, *, /, %
- ⚡ **Bitwise Operations** - Support for &, |, ^, ~, <<, >>, >>>
- 📋 **One-Click Copy** - Copy any result with a single click
- 🎨 **VS Code Native** - Seamless integration with VS Code theme
- ⚡ **Quick Access** - Activity Bar icon, Editor Toolbar, or hotkey

## Usage

### Quick Start
1. Click the BBT icon in the Activity Bar (left sidebar)
2. Or press `Ctrl+Shift+B` (Windows/Linux) / `Cmd+Shift+B` (Mac)
3. Or use Command Palette: `Byte Bit Tool: Open`

### Examples
- `42` → DEC: 42, HEX: 0x2A, BIN: 101010
- `0xFF` → DEC: 255, HEX: 0xFF, BIN: 11111111
- `0b1010` → DEC: 10, HEX: 0xA, BIN: 1010
- `0xFF + 0b1010` → Calculate expressions
- `42 << 2` → Bitwise left shift
- `0xFF & 0x0F` → Bitwise AND

## Installation

### From VS Code Marketplace (Coming Soon)
1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "Byte Bit Tool"
4. Click Install

### Manual Installation
1. Download the `.vsix` file
2. In VS Code: Extensions → ... → Install from VSIX
3. Select the downloaded file
